# Auth란?
보통 Auth를 `인증`이라는 단어로 부른다.

# Authentication VS Authorization

## Authentication
* 인증
* 로그인을 생각하면 된다.

## Authorization
* 인가
* 로그인 후 사용자가 일반 유저인지, 슈퍼유저(관리자)인지 등을 구분하는 것을 생각하면 된다.
